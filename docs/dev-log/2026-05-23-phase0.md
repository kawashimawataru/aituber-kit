## 0-1: Node 24 セットアップ手順（Mac / Homebrew）

### 環境確認

```bash
node -v   # v24.x.x になっていること
cat .nvmrc  # 24 と表示されること
```

### nvm を使う場合

```bash
nvm install 24
nvm use 24
node -v  # v24.x.x
```

### Homebrew + vips がある環境（sharp エラー対策）

```bash
export SHARP_IGNORE_GLOBAL_LIBVIPS=1
npm install
```

`~/.zshrc` に永続化する場合:

```bash
echo 'export SHARP_IGNORE_GLOBAL_LIBVIPS=1' >> ~/.zshrc
source ~/.zshrc
```

### 確認コマンド

```bash
npm run dev
# → http://localhost:3000 で VRM/Live2D が表示されること
```

### 注意

- Node 22 でも起動はするが `.nvmrc` 準拠の 24 を推奨
- Windows / Linux は vips の問題が発生しにくいため `SHARP_IGNORE_GLOBAL_LIBVIPS` は不要なことが多い
