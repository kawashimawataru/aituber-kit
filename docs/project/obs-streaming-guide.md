# OBS 共演配信 構成ガイド

## 概要

AIキャラクター（AITuberKit）と人間配信者が同一 OBS プロジェクト内で共演する際の
シーン・ソース・音声ルーティングの推奨構成を示す。

---

## 推奨シーン構成

```
シーン: 共演-メイン
├── ソース: AIキャラクター（ブラウザソース）
├── ソース: 人間カメラ（映像キャプチャ）
├── ソース: ゲーム画面（映像キャプチャ or ウィンドウキャプチャ）
└── ソース: コメント欄（ブラウザソース: わんコメ or 配信サービス）

シーン: AIのみ
├── ソース: AIキャラクター（ブラウザソース）
└── ソース: ゲーム画面

シーン: 人間のみ
├── ソース: 人間カメラ
└── ソース: ゲーム画面
```

---

## ブラウザソース設定（AIキャラクター）

| 項目           | 推奨値                                              |
| -------------- | --------------------------------------------------- |
| URL            | `http://localhost:3000`                             |
| 幅             | 1920                                                |
| 高さ           | 1080                                                |
| FPS            | 30                                                  |
| CSS            | `.bg-white { background: transparent !important; }` |
| シャットダウン | チェックしない（常時起動）                          |
| リフレッシュ   | チェックしない                                      |

> **透過背景を使う場合**: AITuberKit の背景を「なし」に設定し、OBS ブラウザソースで
> 「OBS とのページ統合を許可」をオンにするとアルファ透過が適用される。

## 音声ルーティング

### 基本構成

```
人間マイク → OBS マイク入力 → 配信出力
AI音声     → AITuberKit Web Audio → ブラウザソース音声 → 配信出力
```

### 音量バランス目安

| ソース           | 推奨音量   | 備考                             |
| ---------------- | ---------- | -------------------------------- |
| 人間マイク       | 0dB 基準   | ノイズゲート推奨                 |
| AI 音声          | -3〜-6dB   | 人間より若干抑えめが自然         |
| ゲーム BGM       | -15〜-20dB | 会話の邪魔にならないよう         |
| SE (stunt/laugh) | -6〜-9dB   | 演出音は目立たせつつ過大にしない |

### OBS 音声フィルタ（人間マイク）

1. **ノイズ抑制** (RNNoise or Speex) — 背景雑音除去
2. **ノイズゲート** — 閾値: -30dB、閉鎖時間: 200ms
3. **コンプレッサー** — 比率: 4:1、閾値: -18dB
4. **ゲイン** — 最終音量調整

## 画面実況機能との連携

AITuberKit の「画面実況」機能は、配信画面のキャプチャを AI が自動解析してコメントする。

### OBS 仮想カメラ経由でキャプチャ

1. OBS → 仮想カメラ → 開始
2. AITuberKit の設定 → 「画面実況」→ 映像ソースで「OBS 仮想カメラ」を選択
3. AIキャラクターが差分を検知すると自動的にコメントを生成

### 直接ウィンドウキャプチャ

- `useScreenCommentary.ts` の `captureElement()` はブラウザ内に描画された要素を対象にする
- OBS 画面は `getDisplayMedia()` API でキャプチャ可能（ブラウザ権限要）

## トラブルシューティング

| 症状                  | 確認箇所                                                      |
| --------------------- | ------------------------------------------------------------- |
| AI音声がOBSに入らない | ブラウザソースの「音声をキャプチャ」がオン確認                |
| 透過が効かない        | OBS ブラウザ統合オン + AITuberKit 背景を透明に設定            |
| AI が無音になる       | `localhost:3000` コンソールで `[speakCharacter]` エラーを確認 |
| stunt SE が聞こえない | `public/assets/reactions/` のファイルが実音源か確認           |
| 画面実況が動かない    | カメラ権限 → ブラウザ設定 → `localhost` にカメラ許可          |
