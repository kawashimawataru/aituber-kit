# わんコメ（OneComme）連携 運用ガイド

## 概要

わんコメ（OneComme）は YouTube / Twitch / ニコニコ等のコメントを一元集約するツール。
AITuberKit はわんコメ WebSocket から視聴者コメントを受信し、AI が応答する。

---

## セットアップ

### 1. わんコメのインストール・起動

1. [わんコメ公式サイト](https://onecomme.com/) からダウンロードしてインストール
2. 配信サービスへのログイン・連携設定
3. わんコメを起動し、コメント取得が開始されることを確認

### 2. AITuberKit 側の設定

AITuberKit の設定 → YouTube タブ:

| 項目                   | 推奨値                           | 備考                       |
| ---------------------- | -------------------------------- | -------------------------- |
| コメント取得方法       | `わんコメ`                       | YouTube Data API の代替    |
| わんコメ WebSocket URL | `ws://localhost:11180/websocket` | わんコメのデフォルトポート |
| コメント取得間隔       | 3秒                              | 低すぎると API 負荷増      |
| 1回のコメント最大数    | 3                                | 同時多発コメント対策       |

### 3. 動作確認

1. わんコメ起動 → AITuberKit 起動 → YouTube タブで「接続」
2. テストコメントを送信し、AIキャラクターが応答するか確認
3. コンソールで `[YouTube] comment received:` ログを確認

---

## 共演配信時のコメント管理

### AI が読まないコメントを除外する

わんコメの「フィルタ」機能でコマンドやスパムを除外:

- `!コマンド` プレフィックスで始まるコメントを除外
- URL を含むコメントを除外
- 特定ユーザー（モデレーター連絡用）を除外

### コメントの優先度設定

| コメント種別     | 対応                                                                 |
| ---------------- | -------------------------------------------------------------------- |
| スーパーチャット | 最優先で読み上げ（AITuberKit 未対応: 手動案内）                      |
| メンバーシップ   | 歓迎メッセージを挟む（preset B の場合は `[stunt:lean_in]` で演出可） |
| 通常コメント     | キューに入れて順次処理                                               |

---

## 共演モード特有の挙動

### 人間配信者が話しているとき

AITuberKit の「共演配信モード」ON 時:

- `humanSpeaking = true` の間は `shouldProactivelySpeak()` が `false` を返す
- アイドル発話が抑制される
- ただしコメントへの即時応答（ユーザーから直接送られたチャット）は引き続き処理される

### コメントが途絶えたとき

- `silenceSec` が `minSilenceSec`（デフォルト 30s）を超えると能動発話が許可される
- アイドルモードの次の発話タイミングで AI が話し始める

---

## [bg:] タグで背景を手動制御

LLM のシステムプロンプト（Preset B 参照）に以下を入れると:

```
[bg:green] でグリーンバック
[bg:default] でデフォルト背景に戻す
```

わんコメのコマンドと組み合わせて視聴者が背景を切り替えることも可能（要: カスタム実装）。

## トラブルシューティング

| 症状                   | 確認箇所                                                                      |
| ---------------------- | ----------------------------------------------------------------------------- |
| わんコメに接続できない | わんコメが起動中か確認、ポート `11180` が塞がれていないか確認                 |
| コメントが届かない     | わんコメ側で対象配信への接続が完了しているか確認                              |
| AI が反応しない        | AITuberKit の YouTube タブで「接続」状態を確認、`chatProcessing` フラグを確認 |
| 大量コメントで詰まる   | 「1回のコメント最大数」を 1〜2 に絞り、わんコメ側でフィルタを強化             |
